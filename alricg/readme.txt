AlriCG - Alpha Version


Dieses ReadMe ist für alle Benutzer die Alricg ausprobieren möchten. Es sei aber dazu
gesagt, dass Alricg im Moment noch NICHT dafür gedacht ist benutzt zu werden und sich 
noch in einer frühen Phase der Entwicklung befindet.
Für mehr Informationen: http://www.alricg.de/

Voraussetzungen:
- Java in der Version 5.0
- Es werden die im Verzeichnis "lib" liegenden Libraries zum Betrieb von Alricg benötigt


Starten von Alricg:
Alricg wird über die Klasse "org.d3s.alricg.controller.ProgAdmin" gestartet, die im Verzeichnis "src" liegt. Wenn alles gut geht, erscheint ein Splash-Screen und die Dateinen werden geladen. Die GUI wird nicht gestartet!
Die JUnit Tests zu Alricg werden über die Klasse "org.d3s.alricg.AllPackageTests" gestartet, die im Verzeichnis "test" liegt.
